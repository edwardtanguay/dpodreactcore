Edward Tanguay's list of realistic API data

This is a list of API data useful for creating realistic coding examples that consume APIs.

Feel free to use this data any way you want.

JSON data and images stored on this site:

  • techItems - my notes on computer languages, software, frameworks, etc.
    • http://edwardtanguay.vercel.app/share/techItems.json
  • flashcards - my flashcards on tech topics, e.g. linux commands, git commands, etc.
    • http://edwardtanguay.vercel.app/share/flashcards.json
  • germanNouns - mostly German tech nouns with article, singular and plural, useful for e.g. die/der/das practice
    • http://edwardtanguay.vercel.app/share/germanNouns.json
  • books - these are general books I have read or am reading, mostly on history, politics, philosohy
    • http://edwardtanguay.vercel.app/share/books.json
    • images: http://edwardtanguay.vercel.app/share/images/books/<idcode>.png
    • e.g. http://edwardtanguay.vercel.app/share/images/books/epicofgilgamesh.png
  • tech books - these are tech books mostly on web development and Linux administration that I am currently reading
    • http://edwardtanguay.vercel.app/share/techBooks.json
    • images: http://edwardtanguay.vercel.app/share/images/techBooks/<idCode>.jpg
    • e.g. http://edwardtanguay.vercel.app/share/images/techBooks/bashCookbook.jpg
  • employees - this is the Northwind employees file with corresponding images created with the site thispersondoesnotexist.com generated with this bash script
    • http://edwardtanguay.vercel.app/share/employees.json
    • images: http://edwardtanguay.vercel.app/share/images/employees/employee_<employeeID>.jpg
    • e.g. http://edwardtanguay.vercel.app/share/images/employees/employee_1.jpg
  • tech books unstructured - these are 10 books which have incongruent data, e.g. properties are missing, not conform, etc. for use in API examples to show how to perform data cleansing
    • http://edwardtanguay.vercel.app/share/techBooksUnstructured.json
    • images: http://edwardtanguay.vercel.app/share/images/techBooks/<idCode>.jpg
    • e.g. http://edwardtanguay.vercel.app/share/images/techBooks/bashCookbook.jpg
  • Spanish verbs unstructured - these are rough notes on Spanish verbs in JSON format, good for demonstrations how to derive clean data out of unstructured data
    • http://edwardtanguay.vercel.app/share/techBooksUnstructured.json

Other APIs and JSON files with realistic data:

  • Gutenberg Books - a searchable catalog of Gutenberg books, includes links to full text of books, images, and zip files
    • https://gutendex.com/books/?search=berlin
  • Northwind Database - this is the 1990s sample database that was shipped with the Microsoft Access database, useful for coding examples since the tables have primary and foreign keys that relate the data together, e.g. customers contains orders which contain products, etc.
    • https://github.com/graphql-compose/graphql-compose-examples/tree/master/examples/northwind/data/json
  • dummyjson.com - a full API that allows GET, POST, PUT, PATCH and DELETE operations
    • https://dummyjson.com

SQLite Databases with realistic data:

  • Northwind Database - this is the SQLite version of the 1990s sample database that was shipped with the Microsoft Access database, useful for coding examples since the tables have primary and foreign keys that relate the data together, e.g. customers contains orders which contain products, etc.
    • northwind_database.zip
